# 🎭 Meme Battle Bot

Телеграм-бот для битвы мемов! Покупай мемов, сражайся с рандомными игроками и вызывай друзей на дуэль.

---

## 📁 Структура проекта

```
meme_battle_bot/
├── bot.py            — точка входа
├── config.py         — токен и настройки
├── database.py       — SQLite база данных
├── memes.py          — каталог мемов и логика боя
├── requirements.txt
└── handlers/
    ├── start.py      — /start, главное меню
    ├── profile.py    — профиль и коллекция
    ├── shop.py       — магазин мемов
    ├── battle.py     — бой с рандомом
    └── duel.py       — дуэль с другом
```

---

## 🚀 Запуск на bothost.ru

### 1. Получи токен бота
- Открой [@BotFather](https://t.me/BotFather) → `/newbot`
- Скопируй токен

### 2. Вставь токен
Открой `config.py` и замени строку:
```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"
```
на свой токен.

### 3. Загрузи на bothost.ru
1. Зайди на [bothost.ru](https://bothost.ru)
2. Создай новый проект → **Python**
3. Загрузи все файлы проекта (zip-архив или через файловый менеджер)
4. В поле **"Команда запуска"** укажи:
   ```
   python bot.py
   ```
5. В разделе **"Зависимости"** или **"requirements.txt"** — файл уже готов, bothost подхватит его автоматически

### 4. Запусти!
Нажми **Start** в панели bothost.ru — бот заработает.

## ⚙️ Настройки (config.py)

```python
START_COINS  = 100   # Монеты при первом входе
DAILY_COINS  = 50    # Ежедневный бонус
BATTLE_REWARD = 30   # Награда за победу над рандомом
DUEL_BET      = 50   # Ставка в дуэли (победитель берёт всё)
```

## ⚔️ Как работает бой

1. Игроки по очереди атакуют
2. Урон = `ATK атакующего - random(0, DEF защитника)`
3. Побеждает тот, кто оставит противника с HP ≤ 0

### Дуэль с другом
1. Создай дуэль → получи ссылку
2. Отправь ссылку другу
3. Друг переходит по ссылке → выбирает мема → бой начинается автоматически
4. Победитель получает `DUEL_BET × 2` монет
